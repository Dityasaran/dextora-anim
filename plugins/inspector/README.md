# data-anim Inspector

A [Chrome extension](https://chromewebstore.google.com/detail/data-anim-inspector/cocpaadjbnbeakonflehnoefjlkhpnmb) for browsing and previewing [data-anim](https://github.com/ryo-manba/data-anim) animations in the browser.

## Development

```bash
pnpm build          # Build script + extension
pnpm build:script   # Build script only
pnpm build:extension # Build extension only
pnpm dev            # Watch mode
```
