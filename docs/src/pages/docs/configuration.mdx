# Configuration

While data-anim works with zero configuration, you can customize global defaults using `window.__dataAnim`.

## Global Options

Set options before the data-anim script loads:

<CodeBlock
  code={'<script>\n  window.__dataAnim = {\n    duration: \'800ms\',\n    easing: \'ease-out\',\n    once: true,\n    offset: 50,\n    debug: false,\n  };\n</script>\n<script src="https://unpkg.com/data-anim@latest/dist/data-anim.js"></script>'}
  lang="html"
  filename="index.html"
/>

## Options Reference

<AttributeTable attributes={[
  { name: 'duration', type: 'string', default: '"600ms"', description: 'Default animation duration for all elements.' },
  { name: 'easing', type: 'string', default: '"ease"', description: 'Default CSS easing function.' },
  { name: 'once', type: 'boolean', default: 'true', description: 'Whether animations play only once by default.' },
  { name: 'offset', type: 'number', default: '0', description: 'Default viewport offset in pixels.' },
  { name: 'debug', type: 'boolean', default: 'false', description: 'Enable debug mode with visual overlays and console logging.' },
  { name: 'disableOnMobile', type: 'boolean', default: 'false', description: 'Disable all animations on mobile devices.' },
]} />

## Attribute Override

Individual `data-anim-*` attributes always override global config:

<CodeBlock
  code={'<script>\n  window.__dataAnim = { duration: \'800ms\', easing: \'ease-out\' };\n</script>\n\n<!-- Uses global: 800ms, ease-out -->\n<div data-anim="fadeIn">Default timing</div>\n\n<!-- Overrides to: 400ms, ease-in -->\n<div data-anim="fadeIn" data-anim-duration="400ms" data-anim-easing="ease-in">\n  Custom timing\n</div>'}
  lang="html"
/>

## Examples

### Fast, snappy animations

<CodeBlock
  code={'window.__dataAnim = {\n  duration: \'400ms\',\n  easing: \'cubic-bezier(0.4, 0, 0.2, 1)\',\n};'}
  lang="js"
/>

### Slow, elegant animations

<CodeBlock
  code={'window.__dataAnim = {\n  duration: \'1200ms\',\n  easing: \'cubic-bezier(0.25, 0.46, 0.45, 0.94)\',\n};'}
  lang="js"
/>

### Disable on mobile globally

<CodeBlock
  code={'window.__dataAnim = {\n  disableOnMobile: true,\n};'}
  lang="js"
/>
