# Attributes

Every data-anim feature is controlled through HTML attributes. No JavaScript configuration needed.

## Required

<AttributeTable attributes={[
  { name: 'data-anim', type: 'string', description: 'The animation name to apply. See Animations for the full list.' },
]} />

<CodeBlock
  code={'<div data-anim="fadeInUp">\n  Animated element\n</div>'}
  lang="html"
/>

## Trigger & Timing

<AttributeTable attributes={[
  { name: 'data-anim-trigger', type: '"viewport" | "hover" | "click" | "focus"', default: 'viewport', description: 'What triggers the animation.' },
  { name: 'data-anim-duration', type: 'string', default: '600ms', description: 'Animation duration (e.g., "400ms", "1.2s").' },
  { name: 'data-anim-delay', type: 'string', default: '0ms', description: 'Delay before animation starts.' },
  { name: 'data-anim-easing', type: 'string', default: 'ease', description: 'CSS easing function (e.g., "ease-out", "cubic-bezier(0.4, 0, 0.2, 1)").' },
]} />

<CodeBlock
  code={'<div\n  data-anim="slideInRight"\n  data-anim-duration="800ms"\n  data-anim-delay="200ms"\n  data-anim-easing="ease-out"\n>\n  Customized timing\n</div>'}
  lang="html"
/>

<AnimationDemo name="slideInRight" />

## Behavior

<AttributeTable attributes={[
  { name: 'data-anim-once', type: '"true" | "false"', default: 'true', description: 'If true, animation plays only once. If false, replays each time the trigger fires.' },
  { name: 'data-anim-offset', type: 'number', default: '0', description: 'Viewport offset in pixels before triggering.' },
  { name: 'data-anim-mirror', type: '"true" | "false"', default: 'false', description: 'Reverse animation when element leaves viewport.' },
]} />

<CodeBlock
  code={'<div\n  data-anim="zoomIn"\n  data-anim-once="false"\n  data-anim-offset="100"\n>\n  Repeats with 100px offset\n</div>'}
  lang="html"
/>

## Stagger

<AttributeTable attributes={[
  { name: 'data-anim-stagger', type: 'string', default: '100ms', description: 'Delay between staggered child animations.' },
  { name: 'data-anim-stagger-pattern', type: '"start" | "end" | "center" | "edges"', default: 'start', description: 'Pattern for stagger order.' },
]} />

<CodeBlock
  code={'<div data-anim-stagger="150ms" data-anim-stagger-pattern="center">\n  <div data-anim="fadeInUp">Item 1</div>\n  <div data-anim="fadeInUp">Item 2</div>\n  <div data-anim="fadeInUp">Item 3</div>\n</div>'}
  lang="html"
/>

## Responsive

<AttributeTable attributes={[
  { name: 'data-anim-disable', type: 'string', description: 'Disable animations at a breakpoint (e.g., "mobile", "tablet").' },
  { name: 'data-anim-mobile', type: 'string', description: 'Override animation for mobile screens.' },
]} />

<CodeBlock
  code={'<!-- Disable on mobile -->\n<div data-anim="slideInLeft" data-anim-disable="mobile">\n  Desktop only animation\n</div>\n\n<!-- Different animation on mobile -->\n<div data-anim="slideInLeft" data-anim-mobile="fadeIn">\n  Slides on desktop, fades on mobile\n</div>'}
  lang="html"
/>

## Advanced

<AttributeTable attributes={[
  { name: 'data-anim-fill', type: '"forwards" | "backwards" | "both" | "none"', default: 'forwards', description: 'CSS animation-fill-mode.' },
  { name: 'data-anim-iteration', type: 'number | "infinite"', default: '1', description: 'Number of times to play the animation.' },
]} />

<CodeBlock
  code={'<div\n  data-anim="pulse"\n  data-anim-iteration="infinite"\n>\n  Infinite pulse\n</div>'}
