How it works

Three steps. That's it.

01

Add the script

Drop one script tag anywhere on your page. No npm install, no bundler, no configuration.

<script src="unpkg…">
02

Write the attribute

Add data-anim="fadeInUp" to any HTML element. Chain delay and duration as needed.

data-anim="zoomIn"
03

Watch it animate

dextora-anim uses IntersectionObserver to trigger animations as elements scroll into view.

scroll → animate

Start in seconds.

One CDN link. No build step. No npm. Works in any HTML file — static sites, CMS templates, legacy apps.
